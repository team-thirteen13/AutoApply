## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px (1rem) | 400 (regular) | 1.5 |
| Label | 14px (0.875rem) | 400 (regular) | 1.4 |
| Heading | 28px (1.75rem) | 700 (bold) | 1.2 |
| Display | 48px (3rem) | 700 (bold) | 1.1 |

**Font mapping:**
- Heading + Display: `font-family: var(--font-space-grotesk), sans-serif` — Space Grotesk
- Body + Label: `font-family: var(--font-inter), sans-serif` — Inter

**Mobile scaling:** Use Tailwind responsive prefixes. Display starts at 36px on mobile, scales to 48px at `md:` breakpoint. Heading starts at 24px on mobile, scales to 28px at `md:` breakpoint.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#ffffff` (white) | Page background, card surfaces, content areas |
| Secondary (30%) | `#1e3a8a` (deep blue) | Hero gradient start, nav text, headings on white |
| Accent (10%) | `#22c55e` (bright green) | Sign-up CTA button only, nav CTA when logged out |

**Accent reserved for:** Sign-up CTA button fill. No other elements use the green accent.

**Hero gradient:** `linear-gradient(135deg, #1e3a8a 0%, #7c3aed 100%)` — full-width, full-height hero section background.

**Gradient shapes:** CSS `radial-gradient` blobs with `filter: blur(80px)` at ~15% opacity, positioned behind text content within the hero section.

**Additional semantic colors:**
| Role | Value | Usage |
|------|-------|-------|
| Destructive | `#ef4444` (red-500) | Destructive actions only (not used in Phase 1) |
| Text primary | `#171717` (neutral-900) | Body text on white backgrounds |
| Text secondary | `#6b7280` (gray-500) | Subtitles, helper text, timestamps |
| Text on dark | `#ffffff` | Text on hero gradient background |
| Nav transparent | `transparent` | Nav background over hero |
| Nav solid | `#ffffff` | Nav background after scroll |

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Sign Up for Free |
| Nav CTA (logged out) | Sign Up |
| Nav CTA (logged in) | Dashboard |
| Hero headline | Build Resumes That Get You Hired |
| Hero subheadline | Create professional, ATS-optimized resumes with AI-powered suggestions tailored to your target roles. |
| Error state | Something went wrong. Please try again or contact support. |
| Destructive confirmation | N/A (no destructive actions in Phase 1) |

**Empty state:** Not applicable — landing page is static content, no data-dependent views.

**Hero fade-in:** Opacity 0 to 1 over 600ms on page load. `prefers-reduced-motion: reduce` disables animation (instant render).

**Primary focal point:** Hero headline text — drawn first by size (48px Display) and centered positioning against the gradient background. Visual hierarchy flows: headline → subheadline → CTA button.

## Interaction Contracts

### Navbar Scroll Behavior

- **Initial state:** Transparent background, white text, over hero gradient
- **Scroll threshold:** 10px — transitions to solid white background, dark text
- **Transition:** `transition-colors duration-300 ease-in-out`
- **Mobile:** Hamburger icon (lucide `Menu`), opens MobileNav overlay
- **MobileNav:** Slide-from-right (`translate-x-full` → `translate-x-0`), 300ms ease-in-out, white background, full viewport height
- **MobileNav backdrop:** Semi-transparent black (`bg-black/50`), fade-in alongside menu

### Hero Animation

- **Fade-in:** `opacity-0` → `opacity-100` over 600ms, triggered on mount
- **Reduced motion:** `@media (prefers-reduced-motion: reduce)` disables animation
- **Gradient blobs:** Static, no animation (pure decorative depth)

### CTA Button Interaction

- **Primary (hero):** Uses existing `Button` component with `variant="gradient"` and `size="lg"`
- **Hover:** Gradient shift (already handled by existing Button gradient variant)
- **Focus ring:** Blue focus ring (existing pattern)
